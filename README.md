# IPV_LLM_research
IPV LLM Classification Pipeline
 
Student Researcher under Dr Biswas, Dr Gunnur, Dr Koyuturk at Case Western Reserve University. Our work focuses on training LLMs with contextual information to classify indicators of intimate partner violence (IPV) and evaluating potential bias in model predictions. The work contributes to ongoing research developing decision-making tools for care providers treating IPV, helping ensure models are accurate and equitable.  


Tools
Languages: Python
Libraries: Transformers, LangChain, Pandas, NumPy
Models: ChatGPT, LLaMA, Qwen
Techniques: Prompt Engineering, retrieval-augmented generation (RAG), bias evaluation
Other Tools: Git, GPU cluster for training
